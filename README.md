2023NKU体系结构课程有关缓存替换策略深入研究，基于Glider实现Falcon替换策略，如需参考报告也请联系repo owner。

## 开启
可以在Linux环境下，首先学习ChampSim如何编写配置文件后，编写好自己的配置文件，然后运行我们编写实现的全自动脚本**allruntrace.sh**，即可自动测试路径下所有trace并输出结果，具体工作逻辑请自行查看脚本内容。
